⚡ QR Code Generator - PindahKeDigital
A high-performance, single-page web utility designed for creating, customizing, and managing QR codes. This tool allows users to generate personalized QR codes with custom branding (colors and logos) and maintains a history of generated assets.

🔗 Visit QR Code Generator | 💼 View My Portfolio

🚀 Core Features
🎨 Customization Engine
Dynamic Color Controls: Full control over Finder Patterns (outer/inner boxes) and Data Modules.

Branding: Seamless logo integration to place your brand identity at the center of the QR code.

Background Styling: Customizable background colors for aesthetic QR designs.

⚙️ System Capabilities
Instant Generation: Real-time generation powered by modern PHP backend logic.

Persistent History: Automatically logs all generated QR codes for quick retrieval and re-download.

Single-Page UX: Clean, intuitive interface built with Tailwind CSS for a frictionless experience.

Downloadable Assets: High-quality PNG export functionality.

🛠️ Technical Stack
Backend: PHP 8.2+ / Laravel 11

Frontend: Tailwind CSS, Alpine.js, Blade

Database: MySQL

Assets: Local storage integration for QR image management

⚙️ Local Installation & Development
To get a local copy up and running, follow these steps:

Clone the Repository

Bash
git clone https://github.com/otnaylus/qrcode.git
cd qrcode
Install Dependencies

Bash
composer install
npm install
Environment Setup

Bash
cp .env.example .env
php artisan key:generate
Database & Storage

Bash
php artisan migrate
php artisan storage:link
Serve Application

Bash
npm run dev
php artisan serve
